Selenium은 webdriver라는 API를 통해 운영체제에 설치된 브라우저를 제어하는 Python 패키지입니다.

Selenium은 웹 자동화 테스트에 사용되는 프레임 워크 입니다.

따라서 셀레니움을 이용한 크롤러는 웹 페이지에서 javascript 렌더링을 통해 생성되는 데이터를 손쉽게 가져올 수 있습니다.

인터넷 브라우저를 통해 크롤링을 하는 개념이라, 실제 보여지는 웹페이지의 전부를 가져올 수 있고, 디버깅 방법또한 직관적입니다.

그러나 웹 브라우저를 실제로 실행시키는 방법이기 때문에 속도도 많이 느리고, 메모리도 상대적으로 많이 차지합니다.

하지만 인터넷 자체에서 크롤링을 진행 할 수 있다는 큰 장점을 이용하여 보다 쉽게 접근하려고 합니다.
